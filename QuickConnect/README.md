## 规则说明
此规则用于使国区群晖强制使用外区的QuickConnect。

并且在套件中心可安装国区被下架的套件，如Synology Chat。

此规则主要解决使用外区QuickConnect时，Synology Chat的推送问题。

并不能帮你通过外区的QuickConnect访问NAS。

为了远程访问，建议使用国区的QuickConnect以获得更好的连接体验。
